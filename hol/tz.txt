Программа Холодильник

Что надо сделать
Нужно написать программу, которая будет:
 - хранить перечень продуктов, их количество и срок годности;
 - позволит добавлять новые продукты в перечень продуктов;
 - позволит получать информацию о содержимом холодильника.


Каждый продукт содержит: 
 - название продукта, 
 - количество продукта,
 - срок хранения (дата, до которой продукт может храниться).
Все продукты хранятся в виде словаря. 
goods = {
    'Пельмени': [
        {'amount': 1, 'expiration_date': datetime.date(2023, 7, 15)},
        {'amount': 2, 'expiration_date': datetime.date(2023, 8, 1)}
    ],
    'Сок': [{'amount':2, 'expiration_date': None}]
} 
Ключ каждого элемента словаря — это название определённого продукта. Значение каждого элемента словаря — это список словарей. 
Каждый вложенный словарь содержит два элемента:
 - amount — количество продукта.
 - expiration_date — срок годности.

Если срок годности не указан, вместо него должно быть указано None.

Операции над данными:
В программе должно быть пять функций:
add() — добавляет продукт в словарь goods;
add_line() — добавляет продукт в словарь goods, преобразуя текстовое описание продукта в структурированные данные;
search() — ищет в словаре goods заданное слово или строку и возвращает список продуктов, в названии которых есть это слово;
amount() — возвращает количество запрошенного продукта;
overdue() — возвращает список просроченных продуктов.
